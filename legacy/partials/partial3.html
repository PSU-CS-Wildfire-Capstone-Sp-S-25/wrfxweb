CU Denver Fire Danger Mapping

This website is a technology demo for the NASA ROSES-#11 project whereby the Wildland fire group demonstrates its technical ability to conduct routine forecasts of fire danger using fire modelling and data assimilation technology developed at CU Denver.

NOTICE

This system is a work in progress. Currently, the cycling system uses NOAA RTMA (Real-Time Mesoscale Analysis) instead of running WRF on the server. Also, only fuel moisture is computed at this time with spread rate and energy output components scheduled for implementation.

Weather forecast

The system computes a 2-day weather forecast for Colorado every day based on initial conditions supplied by NOAA (NAM_218 North American Mesoscale 12km grid). WRF-SFIRE is run in a three domain configuration as shown on the map below. The outermost domain exists to resolve synoptic weather structures that affect weather in the domain of interest - Colorado.

Observation ingest

The system regularly ingests observations from Remote Automatic Weather Stations (RAWS) and satellite fire detections.

The RAWS are retrieved hourly from the Mesowest servers of the University of Utah web API and are fed into a fuel moisture data assimilation algorithm.

The fire detections are obtained from the US Forest Service Active fire maps website and are directly displayed on the map.
